Függvények — feladatgyűjtemény

Három szint: haladj sorban, vagy ugorj a szintedre. A végeredmény minden feladatnál lenyitható — előbb számolj, csak utána nézd meg!

AlapKözép NehézJoker

🟢 Alapszint

1.Alap

Adott az \(A = \{ 1,2,3,4\}\) és a \(B = \{ a,b,c,d\}\) halmaz. Az alábbi relációk közül melyek függvények \(A \rightarrow B\) típusúak? Indokolj! a) \(\rho_{1} = \{\left( 1,b \right),\left( 2,a \right),\left( 3,d \right),\left( 4,c \right)\}\); b) \(\rho_{2} = \{\left( 1,a \right),\left( 2,c \right),\left( 4,d \right)\}\); c) \(\rho_{3} = \{\left( 1,a \right),\left( 2,a \right),\left( 3,a \right),\left( 4,a \right)\}\); d) \(\rho_{4} = \{\left( 1,a \right),\left( 2,b \right),\left( 2,c \right),\left( 3,d \right)\}\).

Végeredmény

a) függvény; b) nem (a \(3\) elemnek nincs képe); c) függvény (konstans); d) nem (a \(2\) elemhez két képelem tartozik).

2.Alap

Adott az \(f:{\mathbb{R}} \rightarrow {\mathbb{R}}\) függvény az \(f\left( x \right) = 3x - 5\) képlettel. Számítsd ki: \(f\left( 0 \right)\), \(f\left( 2 \right)\), \(f\left( - 1 \right)\), \(f\left( \frac{1}{3} \right)\) és \(f\left( a \right)\).

Végeredmény

\(f\left( 0 \right) = - 5\), \(f\left( 2 \right) = 1\), \(f\left( - 1 \right) = - 8\), \(f\left( \frac{1}{3} \right) = - 4\), \(f\left( a \right) = 3a - 5\).

3.Alap

Adott az \(f\left( x \right) = \frac{x}{2} - 3\) függvény. Számítsd ki: \(f\left( 4 \right)\), \(f\left( - 6 \right)\), \(f\left( 2a \right)\) és \(f\left( b + 6 \right)\).

Végeredmény

\(f\left( 4 \right) = - 1\), \(f\left( - 6 \right) = - 6\), \(f\left( 2a \right) = a - 3\), \(f\left( b + 6 \right) = \frac{b}{2}\).

4.Alap

Az \(f:A \rightarrow B\) függvényt a \(f = \{\left( 1,3 \right),\left( 2,3 \right),\left( 3,5 \right),\left( 4,7 \right)\}\) rendezett párok adják meg, ahol \(A = \{ 1,2,3,4\}\). Add meg az értelmezési tartományt \(D_{f}\) és az értékkészletet \(R_{f}\). Injektív-e a függvény?

Végeredmény

\(D_{f} = \{ 1,2,3,4\}\), \(R_{f} = \{ 3,5,7\}\). Nem injektív, mert \(f\left( 1 \right) = f\left( 2 \right) = 3\).

5.Alap

Az \(f\left( x \right) = 2x + 1\) függvényt a \(C = \{ 0,1,2,3\}\) halmazon értelmezzük. Írd fel a függvényt rendezett párok halmazaként, és add meg az értékkészletét!

Végeredmény

\(f = \{\left( 0,1 \right),\left( 1,3 \right),\left( 2,5 \right),\left( 3,7 \right)\}\), \(R_{f} = \{ 1,3,5,7\}\).

6.Alap

Adott az \(f\left( x \right) = 2x - 1\) és a \(g\left( x \right) = x + 4\) függvény. Határozd meg a \(\left( f \circ g \right)\left( x \right)\) és a \(\left( g \circ f \right)\left( x \right)\) függvényt!

Végeredmény

\(\left( f \circ g \right)\left( x \right) = 2x + 7\), \(\left( g \circ f \right)\left( x \right) = 2x + 3\).

7.Alap

Adott az \(f\left( x \right) = 3x\) és a \(g\left( x \right) = x - 2\) függvény. Határozd meg: a) \(\left( f \circ g \right)\left( x \right)\); b) \(\left( g \circ f \right)\left( x \right)\); c) \(\left( f \circ f \right)\left( x \right)\); d) \(\left( g \circ g \right)\left( x \right)\).

Végeredmény

a) \(3x - 6\); b) \(3x - 2\); c) \(9x\); d) \(x - 4\).

8.Alap

Adott az \(f\left( x \right) = x + 1\), a \(g\left( x \right) = 2x\) és a \(h\left( x \right) = x - 3\) függvény. Határozd meg: a) \(\left( f \circ g \right)\left( x \right)\); b) \(\left( g \circ h \right)\left( x \right)\); c) \(\left( h \circ f \right)\left( x \right)\).

Végeredmény

a) \(2x + 1\); b) \(2x - 6\); c) \(x - 2\).

9.Alap

Vizsgáld meg, injektív, szürjektív, illetve bijektív-e az \(f:A \rightarrow B\) függvény, ha \(A = \{ 1,2,3\}\), \(B = \{ p,q,r,s\}\) és a) \(f\left( 1 \right) = p,f\left( 2 \right) = q,f\left( 3 \right) = p\); b) \(g:A \rightarrow A\), \(g\left( 1 \right) = 2,g\left( 2 \right) = 3,g\left( 3 \right) = 1\).

Végeredmény

a) \(f\) nem injektív (\(f\left( 1 \right) = f\left( 3 \right)\)), nem szürjektív (\(s\) nincs képként) → nem bijektív; b) \(g\) injektív és szürjektív → bijektív.

10.Alap

Határozd meg az \(f\left( x \right) = 2x - 6\) függvény inverzét, \(f^{\left( - 1 \right)}\left( x \right)\)!

Végeredmény

\(f^{\left( - 1 \right)}\left( x \right) = \frac{x}{2} + 3\).

11.Alap

Határozd meg az \(f\left( x \right) = \frac{- x}{3} + 2\) függvény inverzét!

Végeredmény

\(f^{\left( - 1 \right)}\left( x \right) = - 3x + 6\).

12.Alap

Adott a \(g\left( x \right) = 2x + 1\) függvény, és tudjuk, hogy \(\left( g \circ f \right)\left( x \right) = 4x - 3\). Határozd meg az \(f\left( x \right)\) függvényt!

Végeredmény

\(f\left( x \right) = 2x - 2\).

13.Alap

A Celsius-fokban megadott hőmérsékletet az \(F\left( x \right) = \frac{9x}{5} + 32\) képlettel váltjuk át Fahrenheit-fokba. a) Hány Fahrenheit-fok a \(20{^\circ}C\)? b) Fejezd ki a képletből a Celsius-fokot! c) Hány Celsius-fok a \(95{^\circ}F\)?

Végeredmény

a) \(68{^\circ}F\); b) \(x = \frac{5\left( F - 32 \right)}{9}\); c) \(35{^\circ}C\).

14.Alap

Adott az \(f\left( x \right) = 5x - 2\) függvény. a) Oldd meg az \(f\left( x \right) = 13\) egyenletet. b) Melyik \(x\)-re lesz \(f\left( x \right) = 0\)?

Végeredmény

a) \(x = 3\); b) \(x = \frac{2}{5}\).

15.Alap

Az \(f\left( x \right) = - 2x + 6\) lineáris függvény grafikonja hol metszi a koordináta-tengelyeket? (Add meg az \(x\)-tengellyel és az \(y\)-tengellyel való metszéspontot.)

Végeredmény

x-tengely: \(\left( 3,0 \right)\); y-tengely: \(\left( 0,6 \right)\).

🟡 Középszint

1.Közép

Adott az \(f\left( x \right) = 2x - 3\) és a \(h\left( x \right) = x^{2} + 1\) függvény. Határozd meg: a) \(\left( f \circ h \right)\left( x \right)\); b) \(\left( h \circ f \right)\left( x \right)\).

Végeredmény

a) \(\left( f \circ h \right)\left( x \right) = 2x^{2} - 1\); b) \(\left( h \circ f \right)\left( x \right) = 4x^{2} - 12x + 10\).

2.Közép

Határozd meg az \(f\left( x \right)\) függvényt, ha \(f\left( x + 2 \right) = 3x - 1\).

Végeredmény

\(f\left( x \right) = 3x - 7\).

3.Közép

Határozd meg az \(f\left( x \right)\) függvényt, ha \(f\left( 2x - 1 \right) = 4x + 3\).

Végeredmény

\(f\left( x \right) = 2x + 5\).

4.Közép

Határozd meg az \(f\left( x \right)\) függvényt, ha \(f\left( \frac{\left( x + 2 \right)}{2} \right) = x - 4\).

Végeredmény

\(f\left( x \right) = 2x - 6\).

5.Közép

Az \(A = \{ 1,2,3,4\}\) halmazon adott az \(f = \{\left( 1,2 \right),\left( 2,4 \right),\left( 3,1 \right),\left( 4,3 \right)\}\) és a \(g = \{\left( 1,3 \right),\left( 2,1 \right),\left( 3,2 \right),\left( 4,4 \right)\}\) függvény. Add meg az \(f \circ g\) és a \(g \circ f\) függvényt rendezett párok halmazaként!

Végeredmény

\(f \circ g = \{\left( 1,1 \right),\left( 2,2 \right),\left( 3,4 \right),\left( 4,3 \right)\}\), \(g \circ f = \{\left( 1,1 \right),\left( 2,4 \right),\left( 3,3 \right),\left( 4,2 \right)\}\).

6.Közép

Az \(A = \{ 1,2,3,4,5\}\) halmazon a \(f = \{\left( 1,4 \right),\left( 2,1 \right),\left( 3,5 \right),\left( 4,2 \right),\left( 5,3 \right)\}\) függvény bijektív. Add meg az inverzét, \(f^{\left( - 1 \right)}\)!

Végeredmény

\(f^{\left( - 1 \right)} = \{\left( 1,2 \right),\left( 2,4 \right),\left( 3,5 \right),\left( 4,1 \right),\left( 5,3 \right)\}\).

7.Közép

Adott az \(f\left( x \right) = 3 - 2x\) és a \(g\left( x \right) = 2x - 1\) függvény. Határozd meg: a) \(f^{\left( - 1 \right)}\left( x \right)\); b) \(g^{\left( - 1 \right)}\left( x \right)\); c) \(\left( f \circ g^{\left( - 1 \right)} \right)\left( x \right)\).

Végeredmény

a) \(f^{\left( - 1 \right)}\left( x \right) = \frac{\left( 3 - x \right)}{2}\); b) \(g^{\left( - 1 \right)}\left( x \right) = \frac{\left( x + 1 \right)}{2}\); c) \(\left( f \circ g^{\left( - 1 \right)} \right)\left( x \right) = 2 - x\).

8.Közép

Bizonyítsd be, hogy az \(f:{\mathbb{R}} \rightarrow {\mathbb{R}}\), \(f\left( x \right) = 2x + 4\) függvény bijektív, majd határozd meg az inverzét!

Végeredmény

Injektív (\(2x_{1} + 4 = 2x_{2} + 4 \Rightarrow x_{1} = x_{2}\)), szürjektív (minden \(y\)-hoz \(x = \frac{\left( y - 4 \right)}{2}\)). \(f^{\left( - 1 \right)}\left( x \right) = \frac{x}{2} - 2\).

9.Közép

Adott az \(f\left( 3x - 1 \right) = 6x - 8\) függvényegyenlet. Határozd meg az \(f\left( x \right)\) és az \(f^{\left( - 1 \right)}\left( x \right)\) függvényt!

Végeredmény

\(f\left( x \right) = 2x - 6\), \(f^{\left( - 1 \right)}\left( x \right) = \frac{x}{2} + 3\).

10.Közép

Adott az \(f\left( 2x + 1 \right) = 4x^{2} + 4x\) függvényegyenlet. Határozd meg az \(f\left( x \right)\) függvényt, majd számítsd ki \(f\left( 3 \right)\) értékét!

Végeredmény

\(f\left( x \right) = x^{2} - 1\), \(f\left( 3 \right) = 8\).

11.Közép

A Fahrenheit–Celsius átváltás képlete \(F\left( x \right) = \frac{9x}{5} + 32\). Fejezd ki a Celsius-fokot a Fahrenheit-fok függvényében (add meg \(F^{\left( - 1 \right)}\)-et), majd számítsd ki, hány \({^\circ}C\) a \(68{^\circ}F\)!

Végeredmény

\(F^{\left( - 1 \right)}\left( x \right) = \frac{5\left( x - 32 \right)}{9}\), \(68{^\circ}F = 20{^\circ}C\).

12.Közép

Adott az \(f\left( x \right) = 4x - 3\) függvény, és tudjuk, hogy \(\left( f \circ g \right)\left( x \right) = 8x + 1\). Határozd meg a \(g\left( x \right)\) függvényt!

Végeredmény

\(g\left( x \right) = 2x + 1\).

13.Közép

Adott a \(g\left( x \right) = 2x + 1\) függvény. Határozd meg az \(f\left( x \right) = g\left( g\left( x \right) \right) - g\left( x \right)\) függvényt!

Végeredmény

\(f\left( x \right) = 2x + 2\).

🔴 Nehéz

1.Nehéz

Határozd meg az \(f\left( x \right)\) függvényt, ha \(f\left( \frac{\left( x - 1 \right)}{2} \right) = 3x - 2\).

Végeredmény

\(f\left( x \right) = 6x + 1\).

2.Nehéz

Adottak az \(f\left( x \right) = 1 - x\), \(g\left( x \right) = \frac{1}{\left( 1 - x \right)}\) és \(h\left( x \right) = \frac{x}{\left( x - 1 \right)}\) függvények (\(x \neq 1\)). Bizonyítsd be, hogy \(g\left( h\left( x \right) \right) = f\left( x \right)\).

Végeredmény

Behelyettesítve \(g\left( h\left( x \right) \right) = \frac{1}{\left( 1 - h\left( x \right) \right)} = \frac{1}{\left( \frac{1 - x}{\left( x - 1 \right)} \right)}\); a nevező \(\frac{\left( \left( x - 1 \right) - x \right)}{\left( x - 1 \right)} = \frac{- 1}{\left( x - 1 \right)}\), így \(g\left( h\left( x \right) \right) = - \left( x - 1 \right) = 1 - x = f\left( x \right)\).

3.Nehéz

Adott az \(f\left( \frac{\left( x - 2 \right)}{2} \right) = \frac{\left( 3x - 5 \right)}{2}\) függvényegyenlet. Határozd meg az \(f\left( x \right)\) függvényt!

Végeredmény

\(f\left( x \right) = 3x + \frac{1}{2}\).

4.Nehéz

Adott az \(f\left( x \right) = 2 - 3x\) és a \(g\left( x \right) = 3x - 1\) függvény. Határozd meg az \(\left( f^{\left( - 1 \right)} \circ g^{\left( - 1 \right)} \right)\left( x \right)\) függvényt!

Végeredmény

\(\left( f^{\left( - 1 \right)} \circ g^{\left( - 1 \right)} \right)\left( x \right) = \frac{\left( 5 - x \right)}{9}\).

5.Nehéz

Adott az \(f\left( 5 - 2x \right) = 4x - 7\) függvényegyenlet. Határozd meg az \(f\left( x \right)\) és az \(\left( f \circ f \right)\left( x \right)\) függvényt!

Végeredmény

\(f\left( x \right) = - 2x + 3\), \(\left( f \circ f \right)\left( x \right) = 4x - 3\).

6.Nehéz

Bizonyítsd be, hogy minden \(f\left( x \right) = ax + b\) alakú függvény (\(a \neq 0\), \(f:{\mathbb{R}} \rightarrow {\mathbb{R}}\)) bijektív, és add meg az inverzét általánosan!

Végeredmény

Injektív és szürjektív \(a \neq 0\) miatt; \(f^{\left( - 1 \right)}\left( x \right) = \frac{\left( x - b \right)}{a}\).

7.Nehéz

Adott az \(f\left( x \right) = \frac{x}{\left( x - 1 \right)}\) függvény (\(x \neq 1\)). Bizonyítsd be, hogy \(\left( f \circ f \right)\left( x \right) = x\) (azaz \(f\) önmaga inverze).

Végeredmény

Behelyettesítve \(\left( f \circ f \right)\left( x \right) = \frac{f\left( x \right)}{\left( f\left( x \right) - 1 \right)}\) a nevező \(\frac{\left( x - \left( x - 1 \right) \right)}{\left( x - 1 \right)} = \frac{1}{\left( x - 1 \right)}\), így az eredmény \(x\).

📝 Gyakorló teszt (dolgozatra) — függvények

1.

Adott az \(f\left( x \right) = 2x - 5\) és a \(g\left( x \right) = 4 - 3x\) függvény. Határozd meg a \(\left( f \circ g \right)\left( x \right)\) és a \(\left( g \circ f \right)\left( x \right)\) függvényt!

Végeredmény

\(\left( f \circ g \right)\left( x \right) = - 6x + 3\), \(\left( g \circ f \right)\left( x \right) = - 6x + 19\).

2.

Az \(A = \{ 1,2,3,4\}\) halmazon adott az \(f = \{\left( 1,3 \right),\left( 2,1 \right),\left( 3,4 \right),\left( 4,2 \right)\}\) és a \(g = \{\left( 1,2 \right),\left( 2,3 \right),\left( 3,1 \right),\left( 4,4 \right)\}\) függvény. Határozd meg: a) \(\left( f \circ g \right)\left( x \right)\); b) \(f^{\left( - 1 \right)}\); c) \(g^{\left( - 1 \right)}\) (rendezett párok halmazaként).

Végeredmény

a) \(f \circ g = \{\left( 1,1 \right),\left( 2,4 \right),\left( 3,3 \right),\left( 4,2 \right)\}\); b) \(f^{\left( - 1 \right)} = \{\left( 1,2 \right),\left( 2,4 \right),\left( 3,1 \right),\left( 4,3 \right)\}\); c) \(g^{\left( - 1 \right)} = \{\left( 1,3 \right),\left( 2,1 \right),\left( 3,2 \right),\left( 4,4 \right)\}\).

3.

Adott az \(f\left( x \right) = - 2x + 6\) függvény. Határozd meg az inverzét, \(f^{\left( - 1 \right)}\left( x \right)\), majd számítsd ki \(f^{\left( - 1 \right)}\left( 2 \right)\) értékét!

Végeredmény

\(f^{\left( - 1 \right)}\left( x \right) = \frac{\left( 6 - x \right)}{2}\), \(f^{\left( - 1 \right)}\left( 2 \right) = 2\).

4.

Határozd meg az \(f\left( x \right)\) függvényt, ha \(f\left( 2x - 1 \right) = 6x + 1\).

Végeredmény

\(f\left( x \right) = 3x + 4\).

5.

Egy taxi viteldíja \(f\left( x \right) = 80x + 250\) dinár \(x\) kilométer után. a) Mennyit fizetünk \(10\) km-ért? b) Hány km-t tettünk meg, ha \(890\) dinárt fizettünk?

Végeredmény

a) \(f\left( 10 \right) = 1050\) dinár; b) \(x = 8\) km.

★Joker

Joker (bónusz). Az \(f\left( x \right) = ax + b\) lineáris függvényről tudjuk, hogy \(f\left( 1 \right) = 5\) és \(f\left( 3 \right) = 11\). Határozd meg az \(a\) és \(b\) értékét, majd add meg az \(f^{\left( - 1 \right)}\left( x \right)\) inverzet!

Végeredmény

\(a = 3\), \(b = 2\), \(f\left( x \right) = 3x + 2\), \(f^{\left( - 1 \right)}\left( x \right) = \frac{\left( x - 2 \right)}{3}\).
